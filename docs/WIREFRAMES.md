## 導航流程圖

```
首頁
  ├─ 查看比分 → 比分頁
  ├─ 舉辦活動 → 建立活動頁
  ├─ 點擊比賽 → 比賽詳情
  └─ 個人資料 → 個人頁

比分頁
  ├─ 點擊比賽 → 比賽詳情
  └─ 下拉刷新 → 重新載入

比賽詳情
  ├─ 按讚/留言 → 更新互動
  └─ 查看球隊 → 球隊資料

探索頁
  ├─ 搜尋 → 搜尋結果
  ├─ 點擊球隊 → 球隊資料
  └─ 點擊球員 → 球員資料

個人頁
  ├─ 設定 → 設定頁
  ├─ 建立活動 → 建立活動頁
  └─ 登出 → 登入頁

活動詳情
  ├─ 報名 → 報名表單
  └─ 管理報名 → 報名名單（主辦者）
```

## 設計原則

1. **一致性**：所有卡片使用相同的圓角、間距和陰影
2. **可讀性**：適當的字體大小和對比度
3. **互動回饋**：按鈕點擊有觸覺回饋
4. **載入狀態**：顯示載入指示器和空狀態
5. **響應式**：支援不同螢幕尺寸
6. **無障礙**：適當的標籤和對比度
